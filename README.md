# ZhiHu

![image](https://github.com/PrideIce/ZhiHu/blob/master/ZhiHu/ZhiHu/Resource/Gif/D9DPQRAHr0.gif)

Swift在一步步发展和完善，期待已久的Swift 5.0也出来了! 

实践是学习的最佳途径，于是在2019年春节期间，决定动手做这个纯Swift项目。

本项目用于学习，欢迎交流，如侵权联系本人删，QQ446118931。

涉及到的知识点：
1. 逆向和砸壳、抓包

2. App架构比较简单，采用较多的成熟第三方框架  
   a. 网络层：Moya(对Alamofire的封装） 
   b. 数据层：SwiftyJson、HandyJson  
   c. UI层：SnapKit、Kingfisher  
   
3. 开发中碰到的比较有意思的点  
   a. 如何实现一个自动平移到屏幕边缘的悬浮按钮？  
   b. 如何拦截H5详情页中底部按钮的JS调用？  
   c. 如何实现一个头部在上滑时自动隐藏的页面？
